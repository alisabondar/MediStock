<!-- GETTING STARTED -->
## Getting Started
Run the following commands in the terminal:
```
npm install
npm run dev
```
Open your web browser and navigate to http://localhost:3000.

### Requirements

An `nvmrc` file is included if using [nvm](https://github.com/creationix/nvm).

- Node 6.13.0
- etc
